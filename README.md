# Todo React App

Разработал SPA-приложение для управления задачами с использованием архитектуры FSD. Реализовал работу с данными через REST API (json-server) на этапе разработки и localStorage в production.

## Демо

https://affectx.github.io/todo-react/

## Функциональность

- CRUD-операции с задачами (добавление, удаление, изменение статуса)
- Поиск и фильтрация задач
- Просмотр детальной страницы задачи без перезагрузки
- Массовое удаление задач
- Динамическое обновление интерфейса

## Работа с данными

- REST API
- Асинхронные запросы (fetch)
- Переключение на localStorage в production
- Синхронизация состояния приложения

№№ Дополнительно

- Реализована клиентская маршрутизация без сторонних библиотек
- Чистая структура проекта и масштабируемая архитектура
- Оптимизация через useMemo, useCallback, React.memo 

- ## Технологии

- React

- SCSS 
- Vite
